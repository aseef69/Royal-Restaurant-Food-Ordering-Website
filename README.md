# 🍽️ Royal Restaurant Website

A modern, responsive restaurant website built using HTML, CSS, and JavaScript. The website showcases the restaurant's menu, chefs, reservation system, customer reviews, and contact information with an elegant and user-friendly design.

---

## 📌 Project Overview

**Royal Restaurant** is a premium restaurant landing page designed to provide customers with an engaging online experience. It features a clean layout, attractive food sections, and responsive design suitable for desktops, tablets, and mobile devices.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/royal-restaurant.git
```

2. Navigate to the project folder:

```bash
cd royal-restaurant
```

3. Open `index.html` in your preferred web browser.

No additional setup is required.
